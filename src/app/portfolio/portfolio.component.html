Project Portfolio

Showcase mechanical builds, prototypes, and supporting software utilities.

Group projects by the problem they solved. Include visuals, design notes, and lessons learned. You can also embed short videos or link to external repos.

  • Mechanical Systems: CAD renders, exploded views, test reports.
  • Automation & Controls: PLC/MCU firmware, dashboards, and tooling.
  • Software Utilities: Scripts or libraries that support your mechanical workflow.

Consider storing project data in JSON under src/app/core/data so new cards can be generated automatically. The docs/ARCHITECTURE.md file explains how to wire new feature modules.
